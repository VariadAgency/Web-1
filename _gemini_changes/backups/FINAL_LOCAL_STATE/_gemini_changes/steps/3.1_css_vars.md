# Step 3.1: CSS Variablen Konsolidierung

**Ziel:** Konsistente Nutzung von Design-Tokens (Farben) im gesamten CSS.
**Betroffene Dateien:** `assets/css/style.css`

## Änderungen
Ersetzung von Hex-Farbcodes durch CSS-Variablen:
- `#dc2626` -> `var(--brand-red)`
- `#fb923c` -> `var(--brand-orange)`
- `#22c55e` -> `var(--brand-green)`
- `#ef4444` -> `var(--brand-red-light)` (Neu definiert)
- `#0a0a0a` -> `var(--bg)`
- `#ffffff` -> `var(--text)` (wo sinnvoll)

## Begründung
Wartbarkeit. Design-Änderungen (z.B. "Marken-Rot etwas dunkler machen") erfordern nur noch eine einzige Änderung im `:root` Block.

## Risiken
- Gering. Syntax-Fehler beim Ersetzen (z.B. doppelte Klammern). Wird durch Skript minimiert.

## Rollback
`cp _gemini_changes/backups/style.css__backup__PHASE3_START.css assets/css/style.css`
