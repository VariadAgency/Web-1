# Step 3.2: Entfernung von !important in #why-ads

**Ziel:** Bereinigung des CSS von erzwungenen Regeln (!important) zur Verbesserung der Wartbarkeit.
**Betroffene Dateien:** `assets/css/style.css`

## Änderungen
- Ersetzen des Selektors `#why-ads` durch `main #why-ads` (höhere Spezifität).
- Entfernen aller `!important` Flags innerhalb dieses Blocks.

## Begründung
`!important` sollte nur im Ausnahmefall genutzt werden. Durch die Erhöhung der Spezifität des Selektors gewinnen die Regeln auf natürlichem Weg gegen die allgemeinen `section` Styles, ohne zukünftige Anpassungen zu blockieren.

## Risiken
- Gering. Falls das Layout springt, war eine andere Regel noch stärker (Rollback möglich).

## Rollback
`cp _gemini_changes/backups/style.css__backup__STEP3.2.css assets/css/style.css`
