# Step 2.5: Meta Tags & SEO

**Ziel:** Optimierung der Darstellung in Suchmaschinen und sozialen Netzwerken.
**Betroffene Dateien:** `index.html`

## Änderungen
Hinzufügen folgender Meta-Tags im `<head>`:
- **Canonical:** Verhindert Duplicate Content.
- **Open Graph (OG):** Steuert die Vorschau auf Facebook, LinkedIn, WhatsApp.
- **Twitter Card:** Steuert die Vorschau auf Twitter/X.
- **Theme Color:** Setzt die Browser-Farbe auf Mobile (Brand-Schwarz).
- **Icons:** Placeholder für Favicons (Apple Touch Icon, etc.).

## Begründung
Eine professionelle Website benötigt diese Tags für eine korrekte Darstellung beim Teilen von Links und für ein gutes Ranking.
